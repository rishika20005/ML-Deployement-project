# 🤖 AutoScribe - Multi-Agent AI Content Generation System

## Overview
AutoScribe is a cloud-native, multi-agent AI system that automates content research, writing, and editing using specialized AI agents.

## Features
- ✅ 3 Specialized Agents: Researcher, Writer, Editor
- ✅ Autonomous workflow (no manual prompting after topic input)
- ✅ Free to use (Google Gemini API free tier)
- ✅ Cloud-deployed on Streamlit Cloud
- ✅ Downloadable article output

## Tech Stack
- **Language:** Python 3.11
- **LLM:** Google Gemini 1.5 Flash (Free API)
- **Frontend:** Streamlit
- **Deployment:** Streamlit Cloud
- **Version Control:** Git/GitHub
